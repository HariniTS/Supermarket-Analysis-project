# Supermarket Sales Analysis (SQL Project)

## Project Overview
This project analyzes supermarket sales data using SQL to extract meaningful business insights such as sales trends, customer behavior, and product performance.

## Dataset
The dataset contains supermarket transaction records including:

- Invoice ID
- Branch
- City
- Customer Type
- Product Line
- Unit Price
- Quantity
- Payment Method
- Date and Time
- Total Sales

## Tools Used
- MySQL
- SQL
- Excel Dataset

## Dataset Preview
![Dataset](dataset_preview.png)

## Business Questions Solved

1. What is the total number of transactions?
2. Which city generates the highest sales?
3. Which payment method is used the most?
4. What is the top selling product line?
5. How do sales vary by month?
6. At what time do most transactions happen?
